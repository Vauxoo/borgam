POS Add Product By Lot/Serial Number

Do away with needless manual tracking on Odoo POS and scan products with Lot/Serial number.

Tracking a product in Odoo POS means entering the lot/serial number of the product everytime you wish to add it to an order.

POS Add Product By Lot/Serial Number allows the Odoo POS user to add the products to an order simply, by scanning the Lot/Serial Numbers associated with that product.

Manually entered lot/serial numbers will be validated on the confirmation. So there is no chance to assign a wrong lot/serial number.

It greatly reduces the efforts of a POS user by eliminating the need to manually enter the Lot/Serial Number for every product.

The module make the whole process of dealing with Lot/Serial Number easy and manageable for an Odoo user.

User can also select the existing lot/serial number manually from the suggestions of the lot/serial numbers associated with that product.

Salient Features
  • The module allows the Odoo user to scan and add the products to the Odoo POS cart/order using Lot/Serial numbers of the products.
  • The lot/serial number can be scanned using barcode scanner.
  • Once a lot/serial number is scanned in Odoo POS, it is automatically assigned the respective lot/serial number in Odoo database as well.
  • The module reduces the effort of assigning a unique barcode number in order to add that product through barcode scanner.
  • The Quantities of products available in the lot are automatically updated whenever a product is validated, added or removed from an order.
  • The system does not allow the Odoo user to add a product quantity higher than that present in the lot to the cart/order. (Refer Below)
  • If the quantity entered to the POS cart/order is greater than number of products in the lot then a pop up will be raised which indicates the insufficient quantity.
  • User can add the product in order manually at that time a suggestion of lot/serial numbers would be shown to user.
  • In case of wrong lot/serial or duplicate serial numbers are assigned/entered by the user a error will be shown by checking the data. (Please refer the below image)

Problems Resolved By This Module

Product Management

By using this module user doesn't need to manually enter the Lot/Serial numbers for every product. Whenever a new product is scanned using Lot/Serial number in POS, the same Lot/Serial number would be assigned to it in POS.

Quantity Of Product In Lot

POS user doesn't need to worry about the quantity of products remaining in the lot as this module will show the quantity of product remaining in the lot, if user enters a value greater than the amount of product in the lot.


Auto Updation of Lot

The Quantity of products remaining in the lot will update automatically within the POS Session. If user enters the lot/serial number manually then it'll also counted in the same lot/serial number if it's exist.

Integrity of Serial Number

As serial number of product is unique, the Odoo user can add only a single product with one serial number.


Easy To Choose Lot/Serial

By using this module user if the user manually enters the lot/serial number then a responsive scrollable suggestions will be shown to select the lot/serial.

Free From Mistakes

User doesn't need to worry about the mistakes because this module provides the lot/serial number validation.

Enable The Traceability For The Product
Select the tracking type :
  • By Lots:

    Products will be considerd as a lot and a lot number is need to assigned for it.

  • By Unique Serial Number:

    Products will be tracked and added by serial number.

Assign A Lot/Serial Number To The Product

Create Lot/Serial Number for a product.

User can manage the quantity of a product in a lot in the Inventory.


Scan Product By Lot/Serial Number

POS user can add the product by scanning the barcode and Lot/Serial Numbers.

If a new product is scanned by a Lot/Serial number then that Lot/Serial number assigned automatically to it.


Lot Empty Indicator

While Adding Products to the Odoo POS cart/order, if the quantity of products in the Lot becomes Zero then a pop up will be shown indicating that there are no products left in the lot.


Quantity Indicator

If the entered quantity of the product is greater than the products remaining in the lot. A popup will be shown with lot number and quantity of product remaining in the lot.

Lot/Serial Scroller

Pos user can choose the product by scrolling over the lot/serial numbers.

Duplicate Serial Number Identifier

If a Pos user enters a serial number more than once than an error would be shown, which indicates that a serial has been assigned more than once.

Lot/Serial Number Validation

If a Pos user enter any wrong or a serial number which is not defined it will be get identified and an error will be shown for the same.

The error consist a checkbox, if user checks the checkbox then the wrong serial number would be assigned to the product.
